shift + стрелка		выделить текст
shift + #           поставить значек # (для комментирования кода за ней)

Ctrl + Z			отменить команду (назад)
Ctrl + Y			воссановить команду (вперёд)

Ctrl + C 			копировать (выделенное)
Ctrl + V 			вставить скопированное

Ctrl + A 			выделить всё
Ctrl + X 			вырезать (выделенное)

Ctrl + / 			закоментировать/раскоментировать линию (или выделенные строки)


Ctrl + Home 		к началу исходника
Ctrl + End 			к концу исходника

Alt + стрелка 		переместить строку (вверх, вниз)
Ctrl + D 			выделить имя которое нужно изменить

Ctrl + shift + k          удалить строку (на которой находится курсор)
Ctrl + shift + пробел     вызвать подсказку в скобках (напрмер в print())

shift +Alt + стрелка(вниз)  копировать строку/выделенные строки (вниз)




Чтобы выйти из программы IPython, вы можете использовать одну из следующих команд:

Введите exit и нажмите Enter.
Введите quit и нажмите Enter.
Кроме того, вы можете просто нажать Ctrl+D (это сигнализирует об окончании ввода).


Очистка терминала:              cls
Выход из Python в терминале:    exit()
Скрыть/показать терминал        Ctrl + J
